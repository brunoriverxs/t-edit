# t-edit

T-shirt editor.

[http://t-edit.herokuapp.com](http://t-edit.herokuapp.com)

![ScreenShot](https://raw.github.com/yayitswei/t-edit/master/t-edit-screenshot.png)

## Usage

Compile the cljs and serve the `public/` directory.

    lein cljsbuild once
    rackup config.ru
    open http://localhost:9292
